O que são HTML, CSS e JavaScript?

HTML (HyperText Markup Language)

HTML é a linguagem de marcação padrão para criar páginas web. Ele define a estrutura do conteúdo da página usando elementos como:

  • <h1> a <h6>: Títulos, onde <h1> é o título mais importante e <h6> é o menos importante.
  • <p>: Parágrafos de texto.
  • <a>: Links para outras páginas ou seções da página.
  • <div>: Divisões ou contêineres para agrupar outros elementos.

CSS (Cascading Style Sheets)

CSS é a linguagem de estilo usada para descrever a apresentação de um documento HTML. Ele controla coisas como:

  • Cor: Mudar a cor do texto ou do fundo.
  • Fonte: Alterar o tipo, tamanho e estilo da fonte.
  • Layout: Definir a posição e o espaçamento dos elementos.

JavaScript (JS)

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas web. Com JS, você pode:

  • Responder a eventos, como cliques de botão.
  • Modificar o conteúdo HTML e CSS dinamicamente.
  • Criar animações e efeitos visuais.
